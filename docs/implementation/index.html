Embedder Microservice

Will call chroma DB, and similar enhancement will be retrieved from DB, and passed to LLM.

Input : retrieves query from queue.

Enhanced query blob from chroma db (if present), else default profile (also retrieved from chroma db) will be attached to the query retrieved from queue and passed to scoring agent

Sanitizer Microservice

input : query from llm

invokes scoring agent to determine score

output : if score < 9 , call LLM. max attempts 3.

if score > 9, call BFF.

maintain correlation id.
